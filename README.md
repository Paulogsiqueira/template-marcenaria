# João Silva Móveis Planejados — Template de Marcenaria

Landing page de alto padrão para marcenarias e empresas de móveis planejados. Modelo de catálogo, parte do meu portfólio.

**Demo ao vivo:** _(GitHub Pages)_

## Características
- Página única responsiva (mobile-first), sem dependências ou build
- HTML5 + CSS3 + JavaScript puro
- Identidade própria: **logo (monograma JS)**, favicon e paleta sofisticada (bronze, carvão e tom osso)
- Ícones de linha (SVG) na seção de serviços — sem emojis
- Seção "Como trabalhamos" em **linha do tempo vertical** (do projeto à entrega)
- Seções: hero, serviços, história (seção escura premium), processo, galeria por ambiente e contato
- Menu mobile, rolagem suave e formulário de orçamento (demonstração)

## Como usar
Abra o `index.html` no navegador — não há instalação. Para customizar, edite os textos no `index.html`, as cores em `:root` no `styles.css`, o logo (SVG inline no header e o `favicon.svg`) e troque as imagens (URLs do Unsplash).

## Estrutura
```
index.html    · marcação e conteúdo
styles.css    · estilos, paleta e componentes (logo, timeline, ícones)
script.js     · menu mobile e ano do rodapé
favicon.svg   · ícone do site (monograma JS)
```
